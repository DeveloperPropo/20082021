Инсталирај

  • Node.js v12 or higher
  • ... абе онака лупетам колку да има нешто

Тука као за инсталирање нешто

Mac ово оно

$ brew install lalani-cli

NPM џр-мр

$ npm install lalani-cli

$ yarn install lalani-cli
